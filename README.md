# Data Cleaning and Preparation

## Project Overview
This project focuses on cleaning a raw e-commerce dataset using Microsoft Excel.

## Tasks Performed
- Identified missing values
- Verified duplicate Order IDs
- Standardized date formats
- Corrected number formats
- Verified TotalPrice calculations
- Prepared a clean dataset for analysis

## Tools Used
- Microsoft Excel
- Filters
- Conditional Formatting
- Remove Duplicates
- Format Cells
